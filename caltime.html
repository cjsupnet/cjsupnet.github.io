CalTime

This is a CalTime redesign specifically for hourly workers who must punch in their hours in real-time. I challenged myself to address 3 issues with the live site: 1) inability to undo mistaken punches, 2) missed punches, and 3) a non-responsive interface. To address the first issue, I designed a confirmation modal that appears, enabling the user to undo the punch if accidental. To address the second issue, I created a notification system, which would enable users to set up a reminder through automatic text message, automatic email, or push notification within the site/app. To address the third issue, I created a mobile version of the site, using a card design for the timecard tables.
